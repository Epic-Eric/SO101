# Contributing

Thanks for your interest in improving SO101! Contributions of all sizes are welcome, from typo fixes to new features.

## Getting Started

1. Fork the repository and create a feature branch from `main`.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   pip install -e ./model
   ```
3. Make your changes with clear, focused commits.

## Pull Requests

- Keep pull requests small and scoped to a single change.
- Update documentation and examples if your change affects them.
- Ensure code is formatted and free of linting errors.
- Describe testing you performed in the pull request template.

## Reporting Issues

If you encounter a bug or have a feature request, please open an issue with:
- A clear description of the problem or proposal.
- Steps to reproduce (for bugs).
- Relevant logs, screenshots, or configuration snippets.

Thank you for helping make SO101 better!
